## 📱 Overview

**Habit Tracker App** is a clean and powerful Flutter application that helps users build and maintain positive habits.
Track your daily progress using a **heatmap visualization**, manage habits easily, and stay consistent with your goals.

---

## ✨ Features

### 🚀 Intro Screen

* Beautiful intro screen with a **Custom Clipper design**
* Smooth onboarding experience

---

### 🏠 Home Screen

* Displays habits with a **heatmap calendar**
* Tracks daily progress visually 📊
* Shows all habits in a clean list

---

### ➕ Add Habit

* Floating Action Button (FAB)
* Add new habits easily
* Simple and user-friendly input

---

### 📝 Manage Habits

* Each habit is displayed in a **ListTile**
* Edit habit ✏️
* Delete habit ❌

---

### 🗑️ Delete All

* One-click button to remove all habits
* Useful for resetting progress

---

### 🌙 Dark Mode

* Full dark mode support
* Theme preference is saved using Hive

---

### 💾 Local Storage

* Uses **Hive** for fast local data storage
* Saves:

    * Habits
    * Dates (for heatmap)
    * Theme mode

---

## 🛠️ Tech Stack

* Flutter
* Dart
* Hive (Local Database)
* Material UI

---

## 🔄 App Flow

Intro → Home → Add Habit → Track Progress → Edit/Delete → Save Locally ✅
